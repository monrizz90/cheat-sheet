Basic HTML Tags

There are so many HTML tags available for use. This cheat sheet should explain how to use some of them.

Name Tag Use
html <html> Tags at the beginning and end of your document.
Head <head> Contains title, meta tags, links to external pages, etc.
Title <title> Text that goes into the title bar.
Body <body> Contains the contents of the document.
Paragraph <p> Allows you to create paragraphs.
Heading <h1-h6> Allows you to create headings of varying importance.
Horizontal Line <hr> Creates a horizontal line.
Line break <br> Creates a break in the current line.
Image <img> Adds an image to the document.
Lists <ol> and <ul> Creates ordered and unordered lists.
Link <a> Adds a link to the document.
